# Visual References

Use this folder for source-of-truth visual targets that should survive chat history.

## `radial-separated-dot-rings.png`

High-quality circular dot field with strict separation. Useful lessons:

- no overlapping nodes
- dark background creates separation and depth
- large white ring forms a readable macro structure without blurring dots
- small accent colors add life without turning the whole field into random noise
- radial placement can imply 3D/depth while staying a flat dot system

## `global-separated-scale-network.png`

Dense global network map with scale variation and clear micro-clusters. Useful lessons:

- many sizes can work when local node separation is preserved
- dark canvas plus saturated category colors keeps dense areas readable
- most tiny nodes remain distinct instead of becoming fog
- labels are rare and attached to high-rank anchors
- network can feel huge without rendering everything as equal-size dots
